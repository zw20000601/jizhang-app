# 小记账本（Web 版）

基于 PRD、设计规范和参考图实现的单页记账应用，包含完整核心功能：

- 首页：总资产、本月收入/支出、预算进度、账户卡片、快捷分类、最近记录、下拉刷新、消息中心入口
- 记录：按月查看、搜索、筛选（全部/支出/收入）、多选、批量删除、批量复制、删除撤销
- 记一笔：支出/收入切换、分类选择、账户选择、备注、日期、数字键盘、编辑已有记录
- 统计：概览/支出/收入/趋势四个子 Tab，含柱状图、环形图、折线图
- 预算：总预算、分类预算、超支提示、按月切换、预算编辑弹层、日均指标
- 设置：账户信息、提醒设置、深色模式、语言设置、CSV 导出、JSON 备份/恢复、立即云同步、仅云恢复、清空数据
- 消息中心：未读角标、消息列表、清空消息、快捷进入提醒设置

## 数据与持久化

- 本地存储：`localStorage`
- 自动云镜像（本地模拟）：每次保存自动更新
- 手动云备份：可手动触发并优先用于云恢复
- 备份恢复优先级：手动云备份 -> 自动云镜像 -> 本地 JSON 文件
- 备份内容包含：账户、交易、预算、设置、消息中心记录
